## 프로젝트 개요

### INSwing이란?
골프 스윙 AI 분석 서비스로, 단순 분석을 넘어 **사용자와 대화하며 성장을 동행하는 골프 코치 플랫폼**

### 현재 상태
- ✅ 영상 업로드 기능
- ✅ AI 프레임/자세 분석 (MediaPipe 기반)
- ✅ 기초 분석 결과 표시
- ⏳ 질문형 코칭 시스템 (개발 예정)
- ⏳ 훈련 계획 및 기록 시스템 (개발 예정)

### 기술 스택
- **Backend**: Node.js + Express + PM2
- **Frontend**: S3 + CloudFront
- **Database**: MySQL
- **Domains**: inswing.ai, api.inswing.ai
- **Infrastructure**: AWS EC2 (43.200.111.14)

## 핵심 가치 및 철학

### 🎯 핵심 문제 인식
기존 골프 분석 서비스들의 한계:
- 영상 업로드 → 수치/그래프/AI 분석 제공까지만
- 사용자 마음속에 남는 질문: **"그래서 나는 뭘 해야 하지?"**
- 수치는 주지만 감동은 없음
- 이해받는 느낌이 없음

### 💡 INSwing의 발견
> **감동은 스윙 수치를 볼 때가 아니라, 질문했을 때 대답을 듣는 순간에 온다.**

### 🔥 서비스 정체성
```
INSwing은 영상을 분석하는 기술이 아니라,
골퍼가 누구인지 이해하고, 질문에 대답해주는 단 하나의 코치입니다.
```

### 핵심 원칙

#### 1. 감동의 순간이 우선
- 데이터를 이해한다 → **사람을 이해한다**
- 분석 결과를 준다 → **해결 방향을 제시한다**
- 정보 제공 서비스 → **관계 맺는 코치**

#### 2. 심리학 기반 설계
- 재미 없으면 사람이 떠난다
- 인정받을 때 더 노력한다
- 스스로 선택했다고 느끼면 애착을 가진다
- 자신을 기억해주는 존재에게 마음을 연다
- 미래의 자신을 보이면 계속 사용한다

#### 3. 균형의 미학
```
천천히, 재미있게, 즐기면서 가되
사용자 만족을 위해서는 목숨 걸 만큼 깊이 고민한다
```

#### 4. 돈과 가치의 균형
- 감동과 성장을 먼저 제공
- 지불의사는 자연스럽게 따라옴
- 강요가 아닌 선택

## 서비스 구조

### 2가지 접근 방식

#### 🟢 기본 모드 (가볍고 빠른 접근)
**대상**: 초보, 바쁜 사람, 부담 없이 시작하고 싶은 사용자

**특징**:
- 영상 업로드만으로 사용 가능
- 자동 분석 + 질문하면 코칭
- 진입 장벽 ZERO

**핵심 경험**:
> "궁금한 것 물어보면 INSwing이 대답해준다."

#### 🔵 정밀 코칭 모드 (사용자가 원할 때 확장)
**대상**: 진지한 골퍼, 체계적 성장을 원하는 사용자

**수집 정보**:
- 골프 경력
- 스코어/핸디
- 목표
- 플레이 환경 (필드/스크린/연습장)
- 선호 스타일

**제공 기능**:
- 맞춤형 코칭
- 훈련 계획
- 느낌 기록
- 성장 추적

**핵심 경험**:
> "나에게 맞는 트레이닝으로 성장하게 해준다."

⚠️ **중요**: 정밀 모드는 강요가 아닌 선택

## 기능 명세

### 핵심 요구사항 정리

#### ✅ 1. 영상 업로드 후 기초 분석
- 현재 구현 완료
- 다음 경험이 INSwing의 핵심

#### ✅ 2. 사용자 질문 시스템
- 스윙에 대해 자유롭게 질문 가능
- "데이터 제공"이 아닌 **대답해주는 서비스**

#### ✅ 3. 질문 대상 선택
- **AI 코치**: 즉시, 무료/기본
- **인간 코치**: 선택적, 확장형

#### ✅ 4. 훈련 계획 생성
- INSwing이 일방적으로 정하지 않음
- 사용자에게 질문하고 **함께 세움**

#### ✅ 5. 실행 추적
- 훈련을 실천하는 과정을 지켜봄
- 기억, 격려, 다음 목표 제시

#### ✅ 6. 사용자 정보 파악
질문 전 간단한 정보 수집:
- 핸디/경력
- 목표
- 연습 환경 (필드/스크린/연습장)
- 연습 가능 빈도

#### ✅ 7. 서비스 접근 방식
- **기본 모드**: 부담 없이, 재미있고 빠르게
- **확장 모드**: 원할 때 깊은 코칭/훈련

#### ✅ 8. 비즈니스 모델
- 감동과 성장을 먼저 제공
- 지불의사는 자연스럽게 따라옴

## 기술 설계

### AI 코칭 시스템 설계

#### 1. AI 코칭 응답 포맷 (INSwing 코어 템플릿)

AI가 어떤 질문을 받든 **4단계 형식**으로 대답:

```
① 현재 문제(원인)를 정확히 설명
② 해결 방법을 제시
③ 느낌/이미지 기반 코칭 (몸으로 느껴지는 표현)
④ 연습 루틴 (실행 가능한 미션)
```

**예시**:
```
① 원인: 다운스윙에서 어깨가 먼저 풀리며 체중이 오른발에 남습니다

② 해결: 왼발을 먼저 밟고 힙을 먼저 돌려 다운스윙을 시작하세요

③ 느낌: "왼발을 밟고 돌린다"는 느낌으로 3번 리허설 후 스윙

④ 루틴: 드라이버 20볼 — 리허설 3회 → 1스윙 반복
```

#### 2. AI 코치 톤/말투 설계

**톤의 특징**:
- 친절 / 과하지 않은 자신감 / 직관적
- 짧고 명확 / 어려운 기술 용어 최소
- 칭찬 + 격려 + 가능성 제시
- 압박 X, 기대 O

**반드시 포함해야 하는 문장 유형**:
- 작은 성취 인정
- 개선 가능성 강조
- 행동을 명확히 안내
- 부담 완화

**절대 금지**:
- ❌ 비판적 말투
- ❌ 비교
- ❌ 지시/명령형
- ❌ 전문 용어 남발

#### 3. AI가 참고해야 할 사용자 정보

| 영역 | 예시 |
|------|------|
| 경력 | 6개월 / 2년 / 10년 |
| 스코어 | 120 / 100 / 90 / 80 / 70 |
| 연습 환경 | 연습장 / 스크린 / 필드 / 시합 |
| 목표 | 비거리 / 방향성 / 드라이버 / 아이언 / 멘탈 |
| 연습 빈도 | 주 1 / 주 2~3 / 거의 매일 |
| 선호 스타일 | 간단한 설명 / 기술적인 설명 |

#### 4. AI 답변 생성 프롬프트 템플릿

```
당신은 인간 심리를 기반으로 동기를 고취시키는 골프 코치입니다.
다음의 사용자 정보를 반영하여 질문에 맞춤 코칭을 제공합니다:

- 골프 경력: {경력}
- 평균 스코어: {스코어}
- 연습 환경: {환경}
- 목표: {목표}
- 연습 빈도: {빈도}
- 선호 스타일: {스타일}

답변 형식은 반드시 다음 4단계를 모두 포함:

① 문제 원인 (짧고 명확하게)
② 해결 방법 (실행 가능한 행동 표현)
③ 느낌/이미지 코칭 (몸으로 느끼는 표현)
④ 연습 루틴 (당장 적용 가능한 미션)

말투 원칙:
- 격려와 인정 포함
- 압박 없이 동기를 자극
- 불필요한 기술 용어 최소화
- "분석"이 아니라 "행동"을 돕는 문장

마지막 한 문장은 사용자가 계속하고 싶어지는 격려로 마무리.
```

### 훈련 계획 자동 생성 로직

#### 1. 입력값

**1️⃣ 사용자 프로필**
- 경력, 스코어, 환경, 목표, 빈도, 선호 스타일

**2️⃣ 스윙 분석 결과**
- 주요 강점/약점 리스트
- 우선 개선 타겟 1~2개

**3️⃣ 코칭 핵심 포인트**
- "이 한 가지를 먼저 잡자" 키워드

**4️⃣ 사용자 제약**
- 연습 가능 횟수
- 피로/부상 여부

#### 2. 출력 형태 (JSON)

```json
{
  "duration_weeks": 2,
  "focus_theme": "피니시 균형 안정",
  "sessions_per_week": 3,
  "sessions": [
    {
      "week": 1,
      "day_label": "월",
      "objective": "피니시에서 왼발 70% 체중 유지",
      "drills": [
        {
          "name": "빈스윙 피니시 3초 유지",
          "balls": 0,
          "reps": 15
        },
        {
          "name": "드라이버 실스윙",
          "balls": 20,
          "focus": "임팩트 후 왼발 밟고 3초 정지"
        }
      ]
    }
  ]
}
```

#### 3. 생성 알고리즘

**① 포커스(Theme) 선정**
- 목표 + 분석결과 조합
- 한 번에 1~2개 테마만

**② 기간 & 빈도 결정**
- 기본 기간: 2주
- 주 1회 → 총 2세션
- 주 2~3회 → 총 4~6세션
- 거의 매일 → 총 8~10세션

**③ 세션 구성**
```
1. 웜업/느낌 만들기
2. 핵심 드릴 (테마와 직접 연결)
3. 실전 적용 (볼 치기)
4. 마무리 & 정리
```

**④ 강도 조절**
- 초보 + 주 1회 + "가볍게" → 30볼 이내
- 중급 + 주 2~3회 + "적당히" → 40~60볼
- 상급 + 거의 매일 + "집중" → 60~80볼

**원칙**: "조금 아쉬울 정도로 끝내야 다음에 또 하고 싶다"

#### 4. 사용자와 함께 계획 세우기

훈련 계획 생성 전 질문:
1. 이번 2주 가장 집중하고 싶은 것?
2. 일주일에 몇 번 연습 가능?
3. 이번 2주는 어떤 느낌으로?

→ **"내가 선택에 참여한 계획"**으로 느끼게
