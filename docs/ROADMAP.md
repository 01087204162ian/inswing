## 1. 개요

INSWING 프로젝트는 "골프 스윙 분석 + 실시간 코칭"이라는 핵심 경험을 기반으로
사용자 개선 체감 → 반복 이용 → 구독 → 확산을 목표로 한다.

### 로드맵의 목적은 다음 3가지이다:

1. 개발 방향의 명확성
2. 반복 가능한 실행 패턴 구축
3. 투자/파트너에게 설명 가능한 근거 정리

## 2. 현재 서비스 상태

| 기능 | 상태 |
|------|------|
| 영상 업로드 → AI 분석 | ✔ 완료 |

| 히스토리 | ✔ 기본형 완료 |

| 모바일 촬영/업로드 | 🟡 동작하나 예외 상황 대응 부족 |
| 소셜 기능 | ❌ 미구현 |
| 구독/결제 | ❌ 미구현 |

**핵심 목표는 실제 사용자가 "계속 쓰고 싶다"고 느끼는 경험 강화다.**

### 🔥 1단계 — "체감형 개선" (2025 Q1)

사용자가 한 번 업로드 → 큰 만족감을 경험하도록 만드는 단계

| 항목 | 범위 |
|------|------|
| 실시간 코칭 MVP → 완성형 | 채팅 로그, 타임라인, 이벤트 |
| 분석 결과 UI 강화 | 메트릭 비교 / 개선 태그 |
| 루틴 UI | 14일 성장 카드 |
| 안정화 | 업로드 실패/대기 처리 / 모바일 대응 |

**🎯 결과**: "한 번 써보면 또 쓰고 싶다" 상태

---

### ⚡ 2단계 — "습관화 / 성장" (2025 Q2)

사용자가 성장 과정을 느끼도록 설계

**항목**

- 루틴 → 배지·레벨 시스템
- 월간 통계 리포트
- 클럽별 진단 및 추천 훈련
- 스윙 유형(슬라이스/훅) 자동 라벨링

**🎯 결과**: "INSWING이 나를 성장시키고 있다" 상태

---

### 🚀 3단계 — "커뮤니티 / 수익화" (2025 Q3)

성장이 → 관계로, 관계가 → 수익으로

**항목**

- AI + 인간 코치 프로 레슨
- 친구/지인 초대 기능
- 코치 프로필 / 레슨 마켓
- 구독제 · 크레딧 · 1:1 레슨 판매

**🎯 결과**: 구독·레슨·클래스 기반의 비즈니스 모델 확립

## 7. 성공 기준 (KPI)

| 지표 | 목표 |
|------|------|
| 첫 업로드 → 두 번째 업로드 비율 | 40%+ |
| 첫 업로드 후 평균 이용 주기 | 7일 이내 |
| 10회 업로드 달성률 | 30%+ |
| 분석 결과 화면 평균 체류 시간 | 2분 이상 |
| 실시간 코칭 기능 이용률 | 50%+ |

**INSWING에서 가장 강한 힘은 "피드백 속도"이다.**

업로드 후 30초 안에 강렬한 피드백 → 지속 이용 → 구독 → 수익.
